# Jeopardy Sound Effects Engine

Drop your bought or downloaded original `.mp3` files into this directory! The game board will automatically play them when the corresponding events happen.

**You must name your files exactly like this:**
- `buzz.mp3` - Plays when a participant clicks their buzzer.
- `correct.mp3` - Plays when you click Correct on the Host Controller.
- `wrong.mp3` - Plays when you click Wrong on the Host Controller.
- `daily_double.mp3` - Plays instantly when a Daily Double tile is clicked on the board.
- `final_jeopardy.mp3` - Plays when you open the Final Jeopardy clue.
